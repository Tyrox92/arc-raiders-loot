# ARC Raiders Loot Helper (Static)

Dies ist eine **bundler-freie** Version. Reines HTML/CSS/JS, läuft als statische Seite.

## Deploy (Vercel ohne Build)
1) ZIP hochladen oder das Verzeichnis verbinden.
2) Projekt-Einstellungen: **Framework = Other**, **Build Command = (leer)**, **Output = /**.
3) Deploy. Die Seite lädt `index.html` im Root.

## Daten erweitern
Bearbeite `items.js` – dort stehen alle Items mit Kategorie/Subkategorie.
